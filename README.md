🚀 AI-Powered Expense Tracker

Welcome to the AI Expense Tracker! 💡 This project is a state-of-the-art personal finance management tool designed to make tracking your spending effortless, insightful, and highly interactive. 📊

✨ Highlights

🎨 Exceptional User Interface: A meticulously crafted, modern UI that is both visually stunning and clutter-free. It provides a seamless and delightful user experience, turning a tedious chore into an engaging habit.

🤖 Intelligent AI Agent: Meet your new financial assistant. Our built-in AI agent allows you to log expenses using natural language, such as stating you spent a certain amount on coffee today ☕, and ask deep questions about your spending habits, like identifying your biggest expense category this month 📈. It offers smart, contextual insights tailored to your data.

🌍 Easily Accessible Components: Accessibility is at our core. All UI components are fully responsive, keyboard-navigable, screen-reader friendly, and designed with high-contrast support to ensure everyone can manage their finances comfortably without barriers.

💰 Robust Functionality: From real-time visual dashboards and budget goal setting to categorized spending breakdowns and data export, this app has everything you need to take absolute control of your wealth.

🧠 How the AI Agent Works

The core differentiator of this project is the conversational AI Agent. Instead of manually filling out tedious forms, you can simply chat to log. 💬 Tell the agent what you bought, and it automatically parses the amount, category, merchant, and date. You can also query your data by asking the agent complex questions, like requesting a comparison of grocery spending across different months 🛒. Additionally, it provides smart advice by sending proactive, AI-driven alerts on where you can save money based on your historical spending trends. 💡

🛠️ Features at a Glance

The tracker includes natural language expense logging alongside interactive, beautiful, and responsive dashboards 📱. All UI components are accessible and WCAG compliant. The application also supports dark and light modes 🌓, custom budget alerts and tracking 🔔, and secure data storage with options to export to CSV or PDF formats 📄.

🚀 Getting Started

To set up the project, you will need Node.js and an API Key for the AI functionality 🔑. You can install the application by cloning the repository, navigating into the project directory, and installing the necessary dependencies 💻. After creating an environment file in the root directory to hold your AI API keys, you can start the development server ⚙️.
